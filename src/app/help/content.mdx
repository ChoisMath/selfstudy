# 포산고 자율학습 사용설명서

이 문서는 학생, 감독교사, 담임교사, 학년관리자, 관리자가 자율학습 출결 시스템을 사용할 때 필요한 기본 흐름을 정리합니다. 화면 구성은 권한에 따라 달라지며, 로그인 후 보이는 메뉴를 기준으로 사용하면 됩니다.

## 설치와 로그인

### 앱 설치

1. 휴대전화 브라우저에서 학교가 안내한 자율학습 시스템 주소로 접속합니다.
2. 브라우저 메뉴에서 홈 화면 추가 또는 앱 설치를 선택합니다.
3. 홈 화면에 생성된 아이콘으로 다시 접속합니다.

### 로그인

학생은 학번 기반 계정으로, 교사는 교사용 계정으로 로그인합니다. 초기 비밀번호를 받은 뒤에는 먼저 비밀번호를 변경해 다른 사람이 계정을 사용할 수 없게 관리합니다.

로그인이 되지 않으면 아이디와 비밀번호 입력 상태를 확인하고, 계속 실패할 때는 담임교사나 관리자에게 계정 상태 확인을 요청합니다.

## 학생 기능

학생 화면에서는 본인의 자율학습 참여 상태와 출결 기록을 확인합니다. 불참이 필요한 경우 사유를 선택해 신청하고, 학년 또는 담임 승인 상태를 확인합니다.

주요 사용 흐름은 다음과 같습니다.

1. 오늘 예정된 자습 시간을 확인합니다.
2. 불참 예정이 있으면 사유와 상세 내용을 입력해 신청합니다.
3. 승인 또는 반려 결과를 확인하고, 반려된 경우 안내에 따라 다시 제출합니다.

## 감독교사 기능

감독교사는 담당 자습 시간에 학생 출결을 확인합니다. 출석, 결석, 사유결석, 방과후 참여 상태를 화면에서 구분해 처리합니다.

### 출석 체크

좌석 배치 화면에서 학생 이름과 반-번호를 확인한 뒤 상태를 변경합니다. 변경한 출결은 저장 후 반영되므로, 전체 확인을 마친 뒤 저장 상태까지 확인합니다.

### 불참신청 승인

감독 대상 시간에 들어온 불참신청은 신청 목록에서 승인 또는 반려할 수 있습니다. 승인하면 출결 기록에도 사유가 연결됩니다.

## 담임교사 기능

담임교사는 담당 학급 학생의 주간, 월간 출결을 확인하고 누락된 사유를 정리합니다. 불참 사유가 없는 결석은 학생 확인 후 사유를 등록합니다.

### 주간과 월간 출결

주간 화면은 빠른 확인에 적합하고, 월간 화면은 누적 현황과 Excel 다운로드에 적합합니다. 월간 자료는 상담, 생활지도, 학급 관리 자료로 활용할 수 있습니다.

### 참여 학생 관리

자율학습 참여 여부가 변경된 학생은 참여 설정에서 반영합니다. 참여하지 않는 학생은 해당 자습 출결 대상에서 제외됩니다.

## 학년관리 기능

학년관리자는 학년 전체의 오늘 출결, 학생 목록, 참여 설정, 좌석 배치, 감독 배정, 월간 출결을 관리합니다.

### 오늘 출결 현황

학년 전체의 오후 자습과 야간 자습 현황을 카드로 확인합니다. 출석, 결석, 사유결석, 방과후 인원을 한 화면에서 비교할 수 있습니다.

### 좌석 배치

좌석 배치에서는 교실별 좌석과 미배정 학생 목록을 함께 확인합니다. 학생을 좌석에 배정한 뒤 저장하면 감독교사 출석 체크 화면에도 같은 배치가 표시됩니다.

미배정 학생 목록에서는 이름 또는 반-번호로 학생을 검색할 수 있습니다.

### 감독 배정

자습 날짜와 시간별 감독교사를 배정합니다. 감독교사가 변경되면 해당 날짜의 감독 화면 접근 권한과 출결 처리 담당자가 함께 바뀝니다.

## 관리자 기능

관리자는 전체 계정과 기초 데이터를 관리합니다. 학년, 학생, 교사, 권한 설정을 정확하게 유지해야 각 사용자 화면이 올바르게 표시됩니다.

### 학생과 교사 관리

학생과 교사는 개별 등록하거나 Excel로 일괄 업로드합니다. 일괄 업로드 전에는 템플릿의 필수 열 이름과 형식을 확인합니다.

<ExcelUploadDemo />

### 권한 관리

관리자는 교사에게 학년관리 권한을 부여하거나 회수할 수 있습니다. 권한 변경 후 해당 교사가 다시 로그인하면 메뉴 구성이 갱신됩니다.

## 문제 해결

| 상황 | 확인할 내용 |
| --- | --- |
| 메뉴가 보이지 않음 | 로그인 계정의 권한이 맞는지 확인합니다. |
| 출결 저장이 되지 않음 | 네트워크 상태를 확인하고 다시 저장합니다. |
| 학생이 목록에 없음 | 학생 활성 상태와 자율학습 참여 설정을 확인합니다. |
| 불참신청이 반영되지 않음 | 승인 상태와 대상 날짜, 자습 시간을 확인합니다. |

문제가 계속되면 오류가 발생한 화면, 날짜, 학생 이름 또는 반-번호를 함께 관리자에게 전달합니다.
